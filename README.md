# ‚≠ê 5 Star Rating Application

This is a simple and interactive 5-star rating web application built using HTML, CSS, and JavaScript. Users can rate an item from 1 to 5 stars and see a message based on their rating.

## Features
- Star-based rating system (1 to 5 stars)
- Displays feedback messages (e.g., "I love it", "It's an average")
- Editable rating with a simple user interface
- Responsive and clean design

## Technologies Used
- HTML
- CSS
- JavaScript
- Font Awesome for star icons

## How to Use
1. Open index.html in your browser.
2. Click on the stars to select your rating.
3. Press the *Rate* button to submit.
4. Click *EDIT* to change your rating.
